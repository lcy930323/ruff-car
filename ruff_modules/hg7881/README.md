# Ruff Driver
这是一个4路驱动板。 
【10P黑色弯排针说明】
1 A1  外接单片机IO口
2 A2  外接单片机IO口
3 B1  外接单片机IO口
4 B2  外接单片机IO口
5 C1  外接单片机IO口
6 C2  外接单片机IO口
7 D1  外接单片机IO口
8 D2  外接单片机IO口
9 +   外接2.5V-12V电压
10 -  外接GND

##接口说明

小车前进

```js
$("#hg7881").turnUp();
```

小车后退

```js
$("#hg7881").turnDown();
```

小车左转

```js
$("#hg7881").turnLeft();
```

小车左转

```js
$("#hg7881").turnRight();
```
